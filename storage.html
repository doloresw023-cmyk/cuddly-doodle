Your Storage

Store and link small files in this repository under assets/. For large files use external hosting and link them here.

Where to add files

  • assets/songs/ — audio files and songs.json
  • assets/images/ — images, screenshots, thumbnails
  • bedrock-addon/ — add-on resources (textures, models)

Quick tips

  1. Keep repo file sizes small (GitHub limits large files). Use external hosts for large media.
  2. Commit files with clear names and update references in the site when adding new assets.
  3. If you want uploads through a UI, I can add a simple server-side uploader (requires hosting).
