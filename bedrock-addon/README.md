Bedrock add-on skeleton

This folder is a minimal skeleton for a Bedrock resource pack. It contains a `manifest.json` with placeholder UUIDs.

To customize:
- Add `textures/`, `models/`, or `behavior_packs/` subfolders as needed.
- Edit `manifest.json` to change the `name`, `description`, and UUIDs (use unique UUIDs).

To install locally:
1. Zip the contents of this folder.
2. Rename the resulting `.zip` to `.mcpack`.
3. Open the `.mcpack` file on a device with Minecraft Bedrock to import the pack.

Notes: This is cosmetic only and contains no gameplay-altering behavior.
